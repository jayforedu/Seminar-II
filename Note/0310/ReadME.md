## 專題一：智慧系統設計與應用－智慧魚缸
本專題旨在透過物聯網與影像辨識技術，打造一個能讓管理更簡單、魚兒生活更舒適的智慧魚缸。
1. 動機與目的
    - 提升便利性與效率：讓日常的魚缸管理更加輕鬆。
    - 穩定生態環境：為魚群營造更穩定的生活空間。
    - 消除不確定性：解決手動餵食與水質維護時常發生的狀況。
2. 系統架構與使用技術
    - 開發技術：使用 Python 語言，結合 OpenCV 模組進行影像處理。
    - 軟體平台：透過 Home Assistant、ESPHome 與 HomeKit 進行系統整合。
    - 硬體設備：以 ESP32 作為核心微控制器，連接各式感測器與繼電器。
3. 系統核心功能
    - 環境感測與控制：
      * 溫度感測器：當水溫低於設定的臨界值時，系統會自動啟動加熱器。
      * 混濁度感測器：偵測到水質混濁度低於標準時，會提醒用戶更換水質。
      * 光度感測器：環境光線不足時，會自動打開 LED 燈光。
    - 自動化系統：
      * 餵食系統：使用者可設定特定魚種的餵食時間，或透過網頁介面隨時手動餵食。
      * 影像辨識與監控：使用監控系統觀察魚隻日常行為，並透過影像辨識技術識別不同魚種，藉此動態調整環境參數。
    - 後臺管理：所有感測資料（水溫、水質、光照度）會傳送至 Home Assistant 操作頁面顯示，並具備即時通知功能。

## 專題二：口罩檢驗管理系統
本研究探討如何利用卷積神經網路（CNN）結合影像預處理技術，應用於產品瑕疵檢測（即口罩品質控制）。
1. 研究背景與方法
   - 結合傳統與深度學習：將車牌辨識常用的「開運算（Opening Operation）」結合 CNN 進行口罩瑕疵檢測。
   - 影像預處理（IP）流程：圖片輸入 $\rightarrow$ 灰階轉換 $\rightarrow$ 腐蝕運算 $\rightarrow$ 膨脹運算 $\rightarrow$ 二值化轉換 $\rightarrow$ 擷取口罩座標 $\rightarrow$ 映射 RGB 圖像 $\rightarrow$ 圖片輸出。
   - 瑕疵分類（共5類）：裁切瑕疵、邊條斷裂、掛繩斷裂、邊條過長、沒有掛繩，以及「正常口罩」。
   - 資料集：共計 6178 張圖片，其中正常口罩為 5057 張，其餘為各類瑕疵圖片。
2. 模型實驗與成效比較
研究共提出了三種結合影像預處理（IP）的模型，並在訓練 60 回合（Epochs）後取得以下成果：
<div align="center">
    
|模型方法|訓練集準確率|驗證集準確率|參數量|平均耗費時間(s)|
|---|---|---|---|---|
|IP-ResNet152V2|97.29%|96.67%|58M|0.055|
|IP-InceptionV3|98.31%|97.56%|21M|0.047|
|IP-Xception   |99.87%|98.21%|21M|0.047|

</div>
結論：在加入影像預處理（IP）後，ResNet152V2、InceptionV3 與 Xception 模型的驗證集準確率均獲得顯著提升，其中以 IP-Xception 的表現最為優異（高達 98.21%）。

## 專題三：改良 YOLOv8 模型之遙測影像目標偵測
本研究旨在優化單階段物件偵測模型（YOLOv8），以應用於高解析度的遙測影像（如智慧交通、災害救援、軍事行動等領域）。
1. 模型改良歷程
研究使用了武漢大學提出的 RSOD 遙測資料集（包含飛機、油灌、高架橋、操場）進行訓練與驗證，並逐步對 YOLOv8n 架構進行改良：
  - YOLOv8n-Bi：引入雙向特徵金字塔網路（Bi-directional Feature Pyramid Network）。
  - YOLOv8n-TFBi：在 Backbone 的最後一個 C2f 區塊替換為 Transformer 區塊，以增加全區域理解能力。
  - YOLOv8n-TFBiCA：進一步添加了座標注意力（Coordinate Attention, CA）區塊，嵌入位置資訊以明確關注特徵圖的特定部分。
2. 實驗結果對比
<div align="center">
    
|模型|mAP@0.5|Precision|Recall|參數量(百萬)|推理時間(ms)|
|---|---|---|---|---|---|
|YOLOv8n         |90.2%|94.9%|89.6%|3.01|3.0|
|YOLOv8n-Bi      |91.7%|95.3%|90.9%|3.02|3.2|
|YOLOv8n-TFBi    |92.4%|95.6%|91.0%|2.86|3.5|
|YOLOv8n-TFBiCA  |94.5%|96.8%|94.1%|2.88|4.0|

</div>
結論：最終提出的 YOLOv8n-TFBiCA 模型在參數量（2.88M）甚至低於原始 YOLOv8n 的情況下，成功將 mAP@0.5 大幅提升至 94.5%，且超越了過往文獻中的 YOLOv5、YOLOv6 與 CA-YOLO 等模型表現。
